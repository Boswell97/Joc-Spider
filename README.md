# Spider-Man Beat-em-up 2D

¡Bienvenido al juego de Spider-Man Beat-em-up 2D!

## Descripción

Este juego es un beat-em-up en 2D ambientado en el universo de Spider-Man. 

## Mecánicas del juego

- **Movimiento:**
  - Utiliza las flechas del teclado para caminar en las direcciones deseadas.
  - Presiona la flecha hacia abajo junto con la flecha izquierda o derecha para arrastrarte.
  - Presiona la flecha hacia arriba para saltar.
  - Corre manteniendo presionada la tecla Shift junto con las flechas de dirección izquierda o derecha.
- **Ataques:**
  - Realiza ataques cuerpo a cuerpo para derrotar a los enemigos.
  - Implementa patadas voladoras con una mecánica de flotación ligera, que te permite planear en el aire y ejecutar ataques desde arriba.
- **Otros:**
  - Explora el entorno y descubre poderes especiales de Spider-Man.
  - Enfréntate a jefes épicos y desafiantes en cada nivel.

## Controles del Personaje

- **Movimiento:** Flechas del teclado (izquierda, derecha, arriba, abajo).
- **Correr:** Flechas del teclado (izquierda o derecha) + Shift izquierdo.
- **Arrastrarse:** Flecha hacia abajo + Flecha izquierda o derecha.
- **Saltar:** Flecha hacia arriba.
- **Ataques:** Teclas de acción (por determinar).

¡Diviértete jugando como Spider-Man y derrotando a los malhechores en este emocionante beat-em-up 2D!
